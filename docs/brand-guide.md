# Cypress Mind — Brand Guide

## Colors
- Hunter Green: `#355E3B`
- Royal Blue: `#4169E1`
- Purple: `#6B3FA0` (use for gradients with Royal Blue)
- Off-White: `#F8FAF9`

## Tone
Professional, clinical, academic, sophisticated.

## Typography
- Headers: Serif font (e.g. Georgia, Playfair Display)
- Body: Sans-serif font (e.g. Inter, DM Sans)

## Design Rules
- No brown or orange
- Minimalist / Mid-century modern aesthetic
- Clean whitespace, structured layouts
